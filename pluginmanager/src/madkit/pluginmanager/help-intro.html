MadKit - Multi Agent Development Kit

Version 4.0

This installer will guide you through the installation of MadKit.

Click the Next button to go to the next install screen. Once all installation parameters have been specified, the Next button changes to Install. Clicking it will then begin the installation process.

The Cancel button will quit the installer without installing anything. If you want to change something about a previous screen, click the Back button.
