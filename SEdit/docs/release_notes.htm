v 3.2 (2001-2004)

There have been many change in SEdit, but this list did not keep trace of it. Let us mention that the Warbot program is based on SEdit for the display area.

Also, SEdit is totally imbedded in MadKit 4

We will start again this version history from now on.

v 2.0b2/b3 "Eclipse"- 05/2000

New Features

  • A documentation, still in its early stage, but which covers most of the features of SEdit (jf)
  • The "Mobile" example formalism.(ol.g, jf)
  • The activate method in structure, nodes and arrows that is called after initial setup of a structure. See the "Logical Gates" example for details.
  • Java look'n'feel icons
  • Startup scripts that automatically build the CLASSPATH (ol.g)
  • Snap to grid feature to stick nodes correctly (jf)
  • A grid may be drawn on the screen (jf).
  • OrgGrapher formalism, which is in its early stage (jf)

Bug fixes

  • Properties are now correctly associated to structures (jf)
  • Graphic properties are now propertly setup when loading a structure. (ol.g)
  • English translation
  • Default graphic properties now work even if there is no graphic class defined.
  • Save and save as commands are now working correctly.
