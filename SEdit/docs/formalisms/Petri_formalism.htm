Petri formalism

The Petri formalism allows the user to describe and simulate Petri nets. A Petri net is made of Places and Transitions which are linked by a set of links.

In the Petri formalism it is possible to describe both standard Petri Nets and Colored Petri Nets, and extended Petri nets by associating predicates and actions to transitions.
