Known bugs

Drawing

  • Diagrams can be printed, but this should be improved
  • Size of the drawing area is computed in a strange way...
  • It is difficult to see when 'iconic' nodes are selected with the mouse.

XML and diagram saving

  • Property values of diagrams are saved even when they have the default value.
  • Generic label and comment properties are not taken into account in arrow description (but they work OK in node description).

Features to be added soon

Generic features

  • Zoom on diagrams
  • Possibility to change the size of nodes
  • Define a GStateIcon to give the possibility to draw different images related to the state of its node.
  • Undo feature to get back to a previous state in the editor...
  • Define a cut and paste that works with other applications (i.e. word processors and drawing applications).
  • Define a generic way to associate graph algorithms to diagrams.
  • Give algorithms to automatically draw diagrams whose node location is not known (automatically generated graphs for instance).
