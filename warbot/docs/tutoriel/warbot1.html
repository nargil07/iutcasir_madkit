Les cercles rouges visualisent les champs de perception de chaque agent et sont obtenus en cliquant sur le bouton detect. Le lancement est obtenu en cliquant sur le bouton .

On peut alors essayer de suivre ce qui se passe, en utilisant les 3 boutons que nous n'avons pas encore vus:

  • le bouton showMessages, qui va permettre de voir les messages échangés entre les agents.
  • le bouton userMessage, qui permet à l'utilisateur d'intervenir,
  • le bouton map, qui va permettre de voir une vue résumée de l'environnement:
